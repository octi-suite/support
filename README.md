# octi suite – Support & Issues

This repository is the **public support and issue tracker** for extensions developed by **octi suite**.

Please use this repository to:
- report bugs
- ask technical questions
- request features

⚠️ This repository does **not** contain any source code.

---

## Supported Products

- **OD Local Fonts** (Joomla extension)
- Additional octi suite extensions may be added over time

When creating an issue, please select the correct product in the issue form.

---

## Before Opening an Issue

Please make sure to include:
- Joomla version
- Extension version
- PHP version
- Template/framework (e.g. Helix Ultimate)
- Clear steps to reproduce (for bugs)

Incomplete issues may be closed with a request for more information.

---

## Security & Privacy

- Do **not** post sensitive data, credentials, or private URLs.
- No personal data is required to open an issue.
